# IP

> IP(인터넷 프로토콜)  

## 역할
- 지정한 IP주소로 데이터 전달
- 패킷이라는 통신 단위 사용

## 패킷 정보
- 출발지 IP, 도착지 IP, 기타...
- 전송할 데이터

## IP 프로토콜의 한계
- 비연결성
    - 패킷을 받을 대상이 없거나 서비스 불능이여도 데이터 전송
- 비신뢰성
    - 중간에 패킷이 사라져도 모름
    - 패킷이 순서대로 안올수도 있음
- 프로그램 구분
    - 같은 IP 서버에서 여러개의 애플리케이션을 사용하면 구분하기 어려움
