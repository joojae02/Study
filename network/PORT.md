- IP 주소 - 목적지 서버 찾는 용 - 아파트 동
- 포트 번호 - 서버 안의 애플리케이션 구분 용 - 아파트 호수
## 포트 번호별
- 0 ~ 65535 할당 가능
- 0 ~ 1023: 잘 알려진 포트, 사용하지 않는 것이 좋음
- FTP - 20, 21
- TELNET - 23
- HTTP - 80
- HTTPS - 443
